# Smart-LPG-Gas-Leakage-Detection-Automatic-Gas-Regulator-Shutoff-using-ESP32-with-Web-Dashboard
IoT-based LPG gas leakage detection system using ESP32, MQ-2 gas sensor, and servo motor to automatically shut off the gas regulator. The system includes a real-time web dashboard for monitoring and SMS alerts using a GSM module to notify users during gas leaks.
